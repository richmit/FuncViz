Detailed Description

template<class treeT, double pointEps>
class mjr::MRPTPoly< treeT, pointEps >

Template Class used to house an poly data generated from a base MRPTree object.

This class is used to transform MRPTree data into polygon data suitable for consumption by visualization software. My personal preference for visualization software are Paraview & messlab, and thus targeted output file formats are VTK (legacy ASCII versions) and PLY (Polygon File Format or Stanford Triangle Format). These files are widely supported by other tools, so my choice of platform shouldn't restrict the usefulness of MRPTree for other users.
