Detailed Description

template<class rt_t, class cc_t>
class mjr::MR_rt_to_cc< rt_t, cc_t >

Template Class used to generate MR_cell_cplx objects from MR_rect_tree objects.

This class is used to transform MR_rect_tree data into geometric data suitable for consumption by visualization software.

Template Parameters
rt_tThe type of the attached MR_rect_tree object
cc_tThe type of the attached MR_cell_cplx object
